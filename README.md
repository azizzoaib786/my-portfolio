# Aziz Zoaib Portfolio Website

Static professional portfolio for azizzoaib.com built with Next.js and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

## Build static site

```bash
npm run build
```

The static export will be generated in the `out` folder.

## Deploy

Recommended: push this project to GitHub and import it into Vercel.
Then connect the custom domain `azizzoaib.com`.

## Replace later

- Add your professional photo in `/public/profile.jpg` if needed.
- Add a downloadable CV as `/public/Aziz-Zoaib-CV.pdf` and link it from the hero button.
